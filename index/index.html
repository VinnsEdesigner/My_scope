GETTING STARTED
1. Tap ▶ START PROBE on splash screen
2. Grant microphone permission
3. Scope begins capturing from mic or AUX jack
4. OSC controls always visible at bottom
5. Open ☰ menu for trigger, calibration, theme
6. Tap SIM tab to use signal simulator
VIEW MODES
OSC — pure waveform
OSC+FFT — waveform + frequency spectrum overlay
OSC+INFO — waveform + diagnostics overlay
SIM — signal simulator with 8 waveform types
SIMULATOR
Select waveform type and adjust FREQ + AMP sliders.
Tap ▶ PLAY to start.
Enable CH2 MIC to compare live mic vs simulator.
Use ☰ SIM SET to access artifacts, channels and source.
TRIGGER SYSTEM
AUTO — always displays
NORMAL — only updates on trigger event
SINGLE — captures once then freezes

RISE ↑ — rising edge crossing upper threshold
FALL ↓ — falling edge crossing lower threshold
AUX JACK SAFETY
⚠️ NEVER connect mains voltage to AUX jack
⚠️ Max safe input: <1Vpp only
