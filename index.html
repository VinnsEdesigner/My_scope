LAB SCOPE PRO — professional mobile oscilloscope.

CH1: Live mic/AUX by default. Optionally load an audio file as the reference signal.

CH2: Synth generator by default. Optionally load an audio file for comparison.

CH3: Auto-computed corrected signal — inverts the detected transfer function from CH1 vs CH2.

OSC channel pills: In SIM mode, use the CH1/CH2/CH3 pills on OSC/FFT/INFO tabs to view each channel individually, full canvas.

Analyser + Bode: Unlock when CH1 has a real external source. Compares CH1 vs CH2, detects HPF, rolloff, AGC, clipping, phase shift.

Session: Your settings, loaded files, and analysis results are saved automatically.

Tip: Calibrate with a known signal before critical measurements.
