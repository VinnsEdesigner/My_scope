LAB SCOPE PRO — professional mobile oscilloscope with FFT spectrum analysis, trigger system, waveform simulation, and signal diagnostics.

OSC Tab: Basic waveform display with AC coupling, zoom/gain controls, and time/volt divisions.

OSC+FFT Tab: Adds frequency spectrum below waveform. Dominant peak highlighted. Useful for harmonic analysis.

OSC+INFO Tab: Full-screen diagnostics panel with signal type detection, RMS, dBFS, trigger status, and calibration factors.

SIM Tab: Multi-channel simulator with synth waveform generator (CH1), live mic capture (CH2), and artifact reconstruction (CH3). File playback and recording supported.

Controls:
• ZOOM: Horizontal zoom (1x - 8x)
• GAIN: Vertical amplification (1x - 10x)
• T/DIV: Manual timebase steps
• V/DIV: Manual voltage steps
• TRIG: Edge-based trigger system (auto/normal/single)
• SNAP: Screenshot capture

Calibration: Use known reference signals (1kHz sine, 1.00Vpp recommended) to correct frequency and voltage measurements.

Simulator: Generates clean reference waveforms (sine, square, triangle, sawtooth, PWM, noise, DC). Capture phone mic response to analyze hardware artifacts (HPF, AGC, clipping, rolloff). Compare CH1 vs CH2 for transfer function analysis.

Tip: For best accuracy, calibrate with known signal before critical measurements.
